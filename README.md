# Covid-Bot
Code written for Datathon,  Data Science Hackathon organized by the UWaterloo Data Science Club. The bot is currently being hosted on Heroku (Nov 21, 2021).

To run the code you need the following packages:
- kaggle
- numpy
- pandas
- pyplot
- discord
